# Custom Scheduler

## Структура проекта
- config.js - настройки приложения 
- logger.js - модуль логирования
- scheduler.js - инициализация и управление задачами с интервалом запуска

## Запуск 
Для запуска проекта необходимо ввести команду
`node scheduler.js`

## Пример вывода работы программы: 
```
[CustomScheduler] scheduler.js started v1.0.0
[CustomScheduler] Task "running" scheduled every 10000ms
[CustomScheduler] running
[CustomScheduler] running
^C
```

## Завершение работы
Для завершения работы проекта необходимо нажать 
`Ctrl + C`
